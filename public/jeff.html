Slide 01 — Your Role & Big Picture

Mission: publish repeatable, useful anti-scam content on Facebook + YouTube to drive awareness and trusted action toward ScamBomb.com.

Your responsibilities
  • Minimum 3–5 Facebook posts daily (you can batch create + schedule)
  • YouTube output is flexible: some weeks one large core video, other weeks many smaller cut-down videos shared to YouTube + Facebook
  • Comment replies are not phase one — we’ll train this later and give you easy tools
  • Keep a simple “done” log (example: “I got 3 videos scheduled today”)
  • Run the pipeline consistently — main KPI right now is Facebook 3x/day minimum
You are NOT responsible for
  • Strategy from scratch
  • On-camera appearances
  • Customer support/billing issues
  • Complex legal responses in comments

Slide 02 — FACEBOOK 101

  • Log in as ScamBomb Page (never personal profile).
  • Create post → paste approved copy → attach media → publish.
  • For video: upload media, add caption, confirm preview, post.
Key metrics
  • Reach
  • Engagement
  • Impressions
  • Shares (gold metric)
  • Comments
Common mistake

Posting from personal profile instead of ScamBomb Page identity.

Slide 03 — YOUTUBE BASICS

  • Go to studio.youtube.com using ScamBomb channel account.
  • Create → Upload video (.mp4 from Canva).
  • Add title, description, thumbnail, playlist, visibility.
  • Publish and verify live status after processing.

Track: views, watch time, CTR, average view duration, subscriber growth.

Slide 04 — CANVA WORKFLOW

Static post
  • Use approved template
  • Edit text only unless instructed
  • Export PNG
  • Keep logo, colors, fonts intact
Video asset
  • Use video template
  • Replace scene text
  • Add approved audio/voiceover
  • Export MP4 1080p

Thumbnail workflow: use template, keep short high-contrast headline, export 1280x720 PNG.

Slide 04B — BRAND GUARDRAILS

  • Colors: ScamBomb navy + gold (+ approved accents only)
  • Typography: headline/body consistency from approved templates
  • Logo: never stretch, recolor, or crop incorrectly
  • Tone: empowering, clear, practical (no panic/fear bait)

Slide 05 — AI TOOLS (PRACTICAL ONLY)

ChatGPT use cases
  • Condense scripts into social copy
  • Prepare future comment-reply drafts for training phase
  • Summarize articles into practical bullets
Gemini use cases
  • Grammar checks
  • Clarity tuning
  • Basic wording validation

Never paste sensitive account or personal data into AI tools.

Slide 06 — GOHIGHLEVEL SCHEDULING

  • Login → Social Planner → New Post
  • Select channel, add text/media, set date/time (Central)
  • Schedule and verify in calendar view

Recommended cadence: batch schedule 2–3 days ahead, stay flexible for breaking scam stories.

Slide 07 — CONTENT PIPELINE SYSTEM

Status flow: Pending → In Progress → Ready for Review → Scheduled → Published

  • Intake: claim tasks in shared queue
  • Produce: create post/video assets from templates
  • Review: run quality gate before scheduling
  • Schedule/Publish: release and verify live
  • Monitor: keep output consistency and log what got done each day

Slide 08 — PRE-PUBLISH QUALITY GATE

Static/Image checks
  • Readable text
  • No spelling errors
  • Correct logo/colors
  • Correct dimensions/file type
  • Links work
Video checks
  • Playback works
  • Audio clear
  • Phone readability
  • No placeholder text
  • Correct title/description/thumbnail

Slide 08B — TROUBLESHOOTING PLAYBOOK

  • Blurry Canva export → increase export quality, use PNG/1080p
  • Scheduled post failed → re-upload media and reschedule
  • YouTube processing delay → wait then re-upload if stuck
  • Spam/abusive comments → not phase one priority; capture examples for later training
  • Missing template/pipeline confusion → notify George/Tushar immediately

Slide 09 — QUICK REFERENCE

  • Copy tools: ChatGPT + Gemini
  • Always include clear CTA toward ScamBomb resources
