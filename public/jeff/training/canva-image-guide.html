🟢 3. Canva Image Guide

How to open a Canva template, edit the text, and export it as an image (PNG)

✅ Goal of this task

You are going to complete this task slowly, carefully, and one step at a time. There is no rush.

Step 1

Open Canva

  1. Open your web browser.
  2. Click the address bar at the top.
  3. Type canva.com.
  4. Press Enter.
👀 What you should see:The Canva homepage or dashboard.
⚠️ If something goes wrong:
  • If you see a login screen, log in.
  • If nothing loads, check your internet.
Step 2

Open Your Template

  1. Click Projects or Your designs.
  2. Find the template you want to use.
  3. It may have a name like Scam Post Template.
  4. Click the template.
👀 What you should see:The design opens. You see an image with text on it.
⚠️ If something goes wrong:
  • If you do not see the template, check Recent designs.
  • If nothing happens, click again.
Step 3

Click the Text to Edit

  1. Move your mouse over the text on the image.
  2. Click directly on the text.
👀 What you should see:A box appears around the text. You may see a blinking cursor.
⚠️ If something goes wrong:
  • If nothing happens, make sure you clicked directly on the words.
  • Try double-clicking.
Step 4

Change the Text

  1. Start typing your new text.
  2. Example: change Scam Alert to Watch Out for This Scam.
  3. Take your time.
👀 What you should see:The words on the image change as you type.
⚠️ If something goes wrong:
  • If you delete too much, click Undo in the top left.
  • If text looks strange, keep typing or ask for help.
Step 5

Adjust Text if Needed

  1. With the text selected, look at the top toolbar.
  2. You may see font size, color, and alignment.
  3. Only change these if needed.
👀 What you should see:The text changes size, color, or position.
⚠️ If something goes wrong:
  • If text disappears, click Undo.
  • If unsure, leave it as-is.
Step 6

Review Your Design

  1. Look at the whole image.
  2. Read the text slowly.
  3. Make sure everything looks correct.
👀 What you should see:A clean image with updated text.
⚠️ If something goes wrong:
  • If something looks off, click the text and fix it.
Step 7

Download the Image

  1. Look at the top right corner.
  2. Click Share.
  3. Click Download.
  4. Find File type.
  5. Choose PNG.
  6. Click Download.
👀 What you should see:The file begins downloading, usually to the Downloads folder.
⚠️ If something goes wrong:
  • If download does not start, click Download again.
  • If file type is wrong, make sure PNG is selected.
Step 8

Find Your Image

  1. Open your Downloads folder.
  2. Look for the newest file.
  3. The file should end in .png.
👀 What you should see:Your finished image file.
⚠️ If something goes wrong:
  • If you do not see it, sort by Most Recent.
  • Try downloading again if needed.

🎉 Done!

You completed this task. Take your time every time. You are doing great.
