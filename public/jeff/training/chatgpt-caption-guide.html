🟢 5. ChatGPT Caption Guide

How to use ChatGPT to generate a simple Facebook caption from a scam topic or script, and copy it into a post

✅ Goal of this task

You are going to complete this task slowly, carefully, and one step at a time. There is no rush.

Step 1

Open ChatGPT

  1. Open your web browser.
  2. Click the address bar.
  3. Type chat.openai.com.
  4. Press Enter.
👀 What you should see:The ChatGPT page with a message box near the bottom.
⚠️ If something goes wrong:
  • If you see a login page, log in.
  • If it will not load, check your internet.
Step 2

Click the Message Box

  1. Move your mouse to the text box at the bottom.
  2. Click inside the box.
👀 What you should see:A blinking cursor.
⚠️ If something goes wrong:
  • If nothing happens, click again inside the box.
Step 3

Type Your Request

  1. Type a simple request.
  2. Example: Write a short Facebook caption about a scam where someone pretends to be a grandchild asking for money.
  3. Do not worry about perfect wording.
👀 What you should see:Your sentence typed in the box.
⚠️ If something goes wrong:
  • If you make a mistake, use Backspace.
  • If it looks messy, that is okay.
Step 4

Send the Request

  1. Look for the arrow or Send button.
  2. Click it. You can also press Enter.
👀 What you should see:ChatGPT starts typing a response.
⚠️ If something goes wrong:
  • If nothing happens, press Enter again.
  • If still nothing happens, refresh and try again.
Step 5

Read the Caption

  1. Wait for ChatGPT to finish.
  2. Read the caption slowly.
👀 What you should see:A short caption about the scam.
⚠️ If something goes wrong:
  • If it is too long, ask for a shorter version.
  • If it is confusing, ask for simpler wording.
Step 6

Ask for a Simpler Version

  1. Click in the message box again.
  2. Type: Make it shorter and simpler.
  3. Press Enter.
👀 What you should see:A shorter and easier caption.
⚠️ If something goes wrong:
  • If it is still too long, ask again.
Step 7

Copy the Caption

  1. Highlight the caption text with your mouse.
  2. Right-click the highlighted text.
  3. Click Copy.
👀 What you should see:The text is copied.
⚠️ If something goes wrong:
  • If right-click does not work, press Command + C on Mac or Ctrl + C on Windows.
Step 8

Go to Facebook

  1. Open a new tab or go back to Facebook.
  2. Make sure you are on the ScamBomb page.
👀 What you should see:The ScamBomb page.
⚠️ If something goes wrong:
  • If you are lost, go back to facebook.com.
Step 9

Paste the Caption into a Post

  1. Click What’s on your mind?
  2. Right-click inside the box.
  3. Click Paste.
👀 What you should see:The caption appears in the post box.
⚠️ If something goes wrong:
  • If nothing appears, press Command + V on Mac or Ctrl + V on Windows.
Step 10

Final Check

  1. Read the caption.
  2. Make sure it looks good.
  3. Fix anything that looks wrong.
👀 What you should see:A clean caption ready to post.
⚠️ If something goes wrong:
  • Click and edit any word that needs fixing.

🎉 Done!

You completed this task. Take your time every time. You are doing great.
