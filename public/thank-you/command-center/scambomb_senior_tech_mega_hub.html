🗄️ PASSWORD MANAGER FIRST-WEEK MAP

🗄️ PASSWORD MANAGER FIRST-WEEK MAP

One step per day. No rush.

  • DAY 1: Download Bitwarden, 1Password, or Dashlane. Create your account.
  • DAY 2: Set your master passphrase and write it down in a safe place.
  • DAY 3: Turn on MFA for the password manager itself.
  • DAY 4: Save your email account password in the manager.
  • DAY 5: Save your banking password. Let the manager generate a new, strong one if needed.
  • DAY 6: Save your social media password.
  • DAY 7: Review and test. Try logging into each account using the manager's auto-fill.

🚨 THE "IS THIS LEGIT?" LOGIN CARD

🚨 THE "IS THIS LEGIT?" LOGIN CARD

Before typing your password, run through this list.

THE FOUR QUESTIONS

  • STEP 1 — DID I GET HERE MYSELF? If you clicked a link in an email or text, stop. Open your browser and type the website address yourself.
  • STEP 2 — DOES THE ADDRESS LOOK CORRECT? Check the URL bar. amazon.com is safe. amaz0n-security.net is a trap.
  • STEP 3 — IS ANYONE RUSHING ME? Urgency is a scammer's favorite tool. Real security teams do not pressure you to "act now."
  • STEP 4 — WHEN IN DOUBT, CLOSE IT OUT. If something feels wrong, close the window. Call the company using a number you already have — not one from the message.

📝 THE 5-MINUTE WI-FI FIX CHECKLIST

📝 THE 5-MINUTE WI-FI FIX CHECKLIST

Run through these in order. Do not skip steps.

  • AIRPLANE MODE OFF — Swiped down, checked for airplane icon, tapped to turn off
  • WI-FI TURNED ON — Settings → Wi-Fi → toggle is green/blue
  • RIGHT NETWORK SELECTED — Name matches my home network, not a neighbor's
  • CLOSE TO ROUTER — Within same room or one room away
  • INTERNET NOT OUT — Checked provider status page or asked neighbor
  • DEVICE RESTARTED — Powered off, waited 30 seconds, powered on
  • ROUTER POWER-CYCLED — Unplugged 2 minutes, plugged modem first, then router
  • BROWSER CACHE CLEARED — Cleared cookies and cached files
  • DIFFERENT BROWSER TESTED — Tried same site in backup browser
  • PROVIDER CALLED — If all above failed, called with account number ready

🧹 THE MONTHLY MAINTENANCE CHECKLIST

🧹 THE MONTHLY MAINTENANCE CHECKLIST

Do these once a month to prevent slowdowns before they start.

  • RESTARTED DEVICE — Full power off and on, not just sleep
  • CLOSED UNUSED BROWSER TABS — Counted tabs; closed everything not actively needed
  • CLEARED BROWSER CACHE — Deleted cached images and cookies
  • EMPTIED RECYCLE BIN / TRASH — Permanently deleted files sitting in bin
  • RAN DISK CLEANUP — Removed temporary files and system cache
  • CHECKED STARTUP PROGRAMS — Disabled anything new that launched automatically
  • REVIEWED DOWNLOADS FOLDER — Deleted old installers and forgotten files
  • RAN ANTIVIRUS SCAN — Quick scan clean; no threats detected
  • CHECKED STORAGE SPACE — Drive has at least 15% free space remaining
  • UPDATED BROWSER — Running latest version; restarted after update
